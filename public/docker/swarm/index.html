Docker Swarm Visualizer ist ein demo image zu anzeige von Services in einem docker swarm. Zum start:

$docker service create \
 --name=viz \
 --publish=8080:8080/tcp \
 --constraint=node.role==manager \
 --mount=type=bind,src=/var/run/docker.sock,dst=/var/run/docker.sock \
 alexellis2/visualizer-arm:latest

Dies wird sicher einige Minuten dauern, da das image erst heruntergeladen werden muss. In der zwischenzeit kann man auf einer zweiten Konsole den Status prÃ¼fen:

````
sudo docker service ls
